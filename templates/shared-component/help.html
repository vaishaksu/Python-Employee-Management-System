An employee management system is a platform where all work-related as well as important personal details of an employee is stored and managed in a secure way. By using this system, you can manage admin activities in an easier and quicker way. Employees are the pillar of any organization and an ideal employee management tool makes a big difference to an organization.

When you run the application, it will automatically directed to the HOME Page. Already existing user can enter the username and password and can be logged in to the account. If the user is admin user, then admin can enter his username and password and can be logged in as admin user in to the account.

When the user logged in to the account, it will directed them to the user home page which will show the basic details of the user which includes email, department, job position. User can click on the "Your Profile" option on the top right corner which shows all the basic details of the user along with the personal details. Edit profile - On the user home page user can click on edit details icon, that will take them to edit user page. User can edit few details such as Personal details, Bank details, Skills and Qualities. To update the other information they need to contact to admin to get the details updated as they do not have rights to edit few details. Signup users Admin can only add the new users to the account because admin have permissions to signup for new users. Change Password Users can change their passwords by clicking "Forgot Password" hyperlink on the Login Page and can easily change their account password.
